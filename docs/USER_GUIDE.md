# 用户操作手册

## 1. PC Web 管理端

访问地址：`http://localhost:16666`（本地 Docker 默认）

### 1.1 登录与退出

**登录**
1. 打开浏览器，输入管理端地址
2. 输入用户名和密码
3. 点击【登录】进入系统首页

默认管理员账号：`admin` / `123456`（首次登录后建议修改密码）

**退出**
1. 点击页面右上角用户头像
2. 选择【退出登录】

### 1.2 图书管理

**新增图书**
1. 左侧菜单【图书管理】→【图书列表】
2. 点击【新增图书】
3. 填写 ISBN、书名、作者、出版社、分类、库存等信息
4. 点击【保存】

> 输入 ISBN 后，系统可自动尝试从网络获取图书元数据。

**上下架**
1. 在图书列表中找到目标图书
2. 点击【上架】或【下架】切换状态

### 1.3 借阅管理

**审批借阅申请**
1. 【借阅管理】→【借阅记录】
2. 筛选状态为"申请中"的记录
3. 点击【通过】或【拒绝】完成审批

**登记归还**
1. 筛选状态为"借阅中"的记录
2. 点击【归还】按钮
3. 系统自动计算是否有逾期罚金

### 1.4 用户与权限

**创建用户**
1. 【系统管理】→【用户管理】
2. 点击【新增用户】，填写信息并分配角色
3. 点击【保存】

**分配角色**
1. 在用户列表中点击【编辑】
2. 在角色选择框中勾选需要分配的角色
3. 点击【保存】

### 1.5 数据统计

【数据统计】→ Dashboard 展示：
- 图书总量、今日借阅量、待审批申请数
- 月度借阅趋势图
- 热门图书排行榜
- 读者借阅活跃度

## 2. 微信小程序用户端

### 2.1 登录

1. 打开微信，进入"图书管理系统"小程序
2. 点击【微信一键登录】，授权获取头像和昵称
3. 首次登录需补充手机号和真实姓名

### 2.2 搜索图书

**关键词搜索**
1. 在首页或搜索页顶部输入书名、作者或 ISBN
2. 点击搜索按钮查看结果

**按分类浏览**
1. 在首页点击分类入口（如"计算机"、"文学"）
2. 浏览该分类下的图书列表

### 2.3 借阅与续借

**借阅图书**
1. 在图书详情页确认有可借库存
2. 点击【立即借阅】
3. 选择借阅天数（默认 30 天）
4. 点击【提交申请】，等待管理员审批

**查看借阅进度**
【我的】→【借阅记录】，状态包括：
- 申请中：等待审批
- 借阅中：已取书，在借阅期限内
- 已归还：已还书
- 逾期：超过归还期限

**续借**
1. 在【借阅记录】中选择"借阅中"的记录
2. 点击【申请续借】
3. 系统自动延长归还日期（每本书最多续借 1 次，延长 15 天）

### 2.4 预约图书

**提交预约**
1. 在图书详情页，如显示"暂无库存"
2. 点击【预约】按钮
3. 图书到货后将按预约顺序通知

**取消预约**
1. 【我的】→【预约记录】
2. 点击【取消预约】

### 2.5 个人中心

【我的】页面可查看：
- 头像、昵称、借阅证号
- 信用积分（初始 100 分，逾期会扣分）
- 当前借阅数量、可借上限

## 3. 常见问题 FAQ

**Q1：忘记管理端密码怎么办？**
联系系统管理员重置密码。

**Q2：图书搜索不到？**
检查关键词是否准确，或尝试 ISBN 搜索。如确实没有，可向管理员反馈采购需求。

**Q3：借阅申请被拒绝的原因？**
- 当前借阅数量已达上限
- 信用积分过低
- 有逾期未还的图书

**Q4：逾期罚金如何计算？**
逾期罚金 = 逾期天数 × 每日罚金率（默认 0.5 元/天），上限 50 元。

**Q5：小程序提示"网络异常"？**
检查手机网络，或确认后端服务是否正常运行。
